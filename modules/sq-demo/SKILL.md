# sq-demo

The deterministic demo portfolio shown until a real account connects.
Synthetic, seeded, offline — these are sciqnt's public figures.

Use it to: demo any view without credentials (`sciqnt --once`, charts,
history, `--json` — everything works on demo data); reproduce docs
figures; test the full pipeline without touching a broker.

`demo_mode` config: `auto` (default — shows only while nothing real is
connected), `on` (always), `off` (never).
