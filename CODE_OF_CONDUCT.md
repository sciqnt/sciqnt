# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
Be kind, assume good faith, attack problems not people.

Enforcement: report via GitHub (issue mentioning @DavideGCosta, or a
private security advisory for sensitive matters). Maintainers may warn,
then temporarily or permanently ban for sustained violations.
