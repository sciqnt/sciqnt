## Entry points

- `sciqnt` (no args, at a TTY) → **interactive home**: banner + portfolio summary + connection status + actions menu (View details / Connect an account / Refresh / Settings / Browse modules / Quit). ^C or Quit exits.
- `sciqnt --once` (or piped) → print the aggregated portfolio once and exit (the non-interactive dump; scripts use this).
- `sciqnt --asof YYYY-MM-DD` → PIT historical view (derived from each broker's CSV history).
- `sciqnt degiro setup --account work` → add a second Degiro account; appears as `degiro:work` in the aggregated view. Omit `--account` for the legacy single-account scheme. Per-account CSV history goes in `data/degiro/work/`.
